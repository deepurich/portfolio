Body Tag most useable tag in HTML. Everything inside the body tag. You add an element or tag show inside the main browser window. The `<body>` opening body tag and `</body>` closing tag you write text, Element, and html tag Show Inside Browser.

**Syntax:**

```html
<body> 
	<h2>Add Tag Here </h2> 
</body>
```
